Graphviz

Graphviz stands for graph visualization software. Graphs are created by describing them in a specifically created language: dot.

How-to

Graphviz - Compile

# dot >> png
cat ./file.dot | dot -Tpng > ./file.png

# dot >> svg
cat ./file.dot | dot -Tsvg > ./file.svg

# dot >> pdf
cat ./file.dot | dot -Tpdf > ./file.pdf

Shapes

digraph D {
  A [shape=diamond]
  B [shape=box]
  C [shape=circle]
  A -> B [style=dashed, color=grey]
  A -> C [color="black:invis:black"]
  A -> D [penwidth=5, arrowhead=none]
}
