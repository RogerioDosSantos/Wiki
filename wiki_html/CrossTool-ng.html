Crosstool-ng

Crosstool-ng is an utility that allow us to build Toolchains.

Download Website: http://crosstool-ng.org/download/crosstool-ng/?sort=modtime&order=desc

Commands

Note: Call those commands on the stage directory and a real linux directory (not a mapped Virtua Box direcory)

ct-ng menuconfig : Call the confguration menu. The result will be stored into the .config file

ct-ng build : Build the .config file.
